# 物归处 (ItemFinder)

**物归处 (ItemFinder)** 是一款致力于解决个人“找不到东西”痛点的个人物品管理应用。通过结合AI图像识别技术，提供极简的物品录入和本地搜索体验。

## 项目特点

- **极简体验**：没有复杂的账号系统，开箱即用。
- **AI 赋能**：集成智谱 GLM-4V-Flash，支持拍照智能录入物品。
- **隐私保护**：所有数据（含位置信息）均保存在设备本地（SQLite），充分保障个人隐私。
- **离线支持**：即使在无网络环境下，也支持手动录入和本地检索，核心功能不断链。

## 技术栈

- 前端：React Native
- 数据库：SQLite (纯本地)
- AI：智谱 GLM-4V-Flash API

## 开发者须知

本项目强制使用 `pnpm` 作为包管理工具，以保证依赖版本一致性和更快的安装速度。Node.js 版本请参考 `.nvmrc`（v18.20.8）。

### 启动项目

```bash
# 1. 安装依赖
pnpm install

# 2. 启动项目
pnpm run start
```
