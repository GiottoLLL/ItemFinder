# Agent 协作规范 (AGENTS.md)

欢迎协助开发 **物归处 (ItemFinder)**！在本项目中，请所有的智能体（AI Agents）及开发者遵循以下基本规范：

## 1. 包管理工具优先级
本项目**强制**使用 `pnpm` 作为唯一的包管理工具。
- **安装依赖**：请使用 `pnpm install <package>` 或 `pnpm add <package>`，绝对**不要**使用 `npm install` 或 `yarn add`。
- **运行脚本**：请使用 `pnpm run <script>`。

## 2. Node 环境
本项目指定 Node.js 版本为 **18.20.8**，详情已写入 `.nvmrc` 文件中。请在执行命令前确认环境是否匹配。

## 3. 技术栈约束
- 框架：**React Native**
- 数据库：本地 **SQLite**
- UI 库/组件：按需引入，但需保持轻量，避免不必要的冗余依赖。

## 4. 编码规范
- **语言**：中文（代码注释、提交记录优先使用中文）。
- **文件命名**：组件文件请使用大驼峰（PascalCase），工具类和普通脚本请使用小驼峰（camelCase）或短横线（kebab-case）。
- 优先做到代码语义化，复杂逻辑需补充“为什么这么做”的注释。
- 请使用 ES6+ 语法。
